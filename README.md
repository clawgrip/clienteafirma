# Firmas PAdES / CAdES para uso en dispositivos móviles

Este proyecto es una simplificación y limpieza de las funcionalidades de firma CAdES y PAdES del proyecto AutoFirma para facilitar su uso en aplicaciones móviles iOS (mediante J2Obc) y Android.

## ¿Por qué es necesario esta modificación

El proyecto AutoFirma presenta un código con una elevada deuda técnica que dificulta la depuración y el entendimiento del código, a la vez que hace más propenso a fallos el uso de J2Obc.
Esta modificación mantiene la base y la funcionalidad original, pero reduce considerablemente la deuda técnica.
